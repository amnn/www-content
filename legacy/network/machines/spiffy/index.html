Spiffy is a Sun UltraSparc Enterprise 1 donated by Warburg Dillon Read in Trinity Term 1999. It currently runs Linux and has a SCSI RAID array attached (also donated by WDR) which holds all the user home directories.

Spiffy became our main user login box over Christmas 1999 after Sticky started showing hardware problems. It also runs our news server, and, since fishy's most recent problems, mail server.
